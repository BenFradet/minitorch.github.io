## Standard Unit Test

- Test for values with given inputs

- PyTest succeeds if no assertions are called
<!-- #endregion -->

```python slideshow={"slide_type": "x"}
def test_relu():
    assert operators.relu(10.0) == 10.0
    assert operators.relu(-10.0) == 0.0

Ideal: Property Test

Test that all values satisfy property

```python slideshow={"slide_type": "x"} def test_relu(): for a in range(0, 1e9): assert operators.relu(a) == a

for a in range(-1e9, 0):
    assert operators.relu(a) == 0.0

```

return new_fn

Higher-order Filter

Extended example

```python slideshow={"slide_type": "x"}

def filter(fn: Callable[[float], bool]) -> Callable[[Iterable[float]], Iterable[float]]: def apply(ls: Iterable[float]): ret = [] for x in ls: if fn(x): ret.append(x) return ret

return apply

```
