```python slideshow={"slide_type": "x"}
class Square(ScalarFunction):
    @staticmethod
    def forward(ctx: Context, x: float) -> float:
        ctx.save_for_backward(x)
        return x * x

    @staticmethod
    def backward(ctx: Context, d: float) -> Tuple[float, float]:
        (x,) = ctx.saved_values
        f_prime = 2 * x
        return f_prime * d

Chain Rule
-----------

Compute derivative from chain

 $$f'_x(g(x)) = g'(x) \times f'_{g(x)}(g(x))$$

<!-- #endregion -->


<!-- #region slideshow={"slide_type": "slide"} -->
Chain Rule
-----------

$$
\begin{eqnarray*}
z &=& g(x) \\
d &=& f'(z) \\
f'_x(g(x)) &=& g'(x) \times d \\
\end{eqnarray*}
$$
<!-- #endregion -->

\(\(log(x)^2\)\)

Example: Chain Rule

$$ begin{eqnarray*} f'(z) &= 2z times 1 \ g'(x) &= 1 / x end{eqnarray*} $$

What is the combination?

\(\(f'_x(g(x))\)\)

Example: Chain Rule

\(\(((x)^2)^2\)\)

$$ begin{eqnarray*} f(z) &= z^2\ g(x) &= x^2\ \ f'(z) &= 2times z\ g'(x) &= 2 times x \ end{eqnarray*} $$

Example: Chain Rule

$$ begin{eqnarray*} f'_x(g(x)) &= 2 times x times 2 times x^2 = 4 x^3\ end{eqnarray*} $$

Tool

*  the derivative with respect to a scalar
*  the function last called on the scalar

We can apply the chain rule through that function.

<!-- #region slideshow={"slide_type": "slide"} -->
Issue
----------

Order matters!

* If we proceed without finishing a variable,
  we may need to apply chain rule multiple times

Desired property: all derivatives for a variable before backward.
<!-- #endregion -->

<!-- #region slideshow={"slide_type": "slide"} -->
Ordering Step
--------------

* Do not process any Variable until all downstream Variables are done.

* Collect a list of the Variables first.
<!-- #endregion -->

<!-- #region slideshow={"slide_type": "slide"} -->
Topological Sorting
---------------------

* [Topological Sorting](http://wikipedia.org/wiki/Topological_sorting)
* High-level -> Run depth first search and mark nodes.
<!-- #endregion -->

<!-- #region slideshow={"slide_type": "slide"} -->
Topological Sorting
---------------------
visit(last)

function visit(node n) if n has a mark then return

for each node m with an edge from n to m do
    visit(m)

mark n with a permanent mark
add n to list
